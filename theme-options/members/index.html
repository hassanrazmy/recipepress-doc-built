Members Related Options

This section has multiple self explaining settings as listed below.

1 - You can show/hide User Naviation in the header.

2 - You can show/hide the reCaptcha on user login/registration popup.

3 - You can show/hide the Recipe Submit button in the header area. It also need the submit page selected in recipe submit options.

4 - You can restrict the admin dashboard access till the specific roles.

5 - You need to select the user profile page to show it in the user navigation menu. For that you need to create a new page and then select the My Profile template from page attributes. After that select the page from here.
